# 簡易単位変換アプリ (Simple Unit Converter)

Pythonで動作する、シンプルで拡張性の高い単位変換アプリケーションです。
ターミナル上で素早く計算できる**CUI版**と、直感的に操作できる**GUI（デスクトップアプリ）版**の2種類を提供しています。

## 🚀 機能特徴

- **2つのインターフェース**: 用途に合わせてCUIとGUIを選んで実行可能。
- **主要な単位に対応**:
  - 長さ: メートル (m) ⇄ フィート (ft)
  - 重さ: キログラム (kg) ⇄ ポンド (lb)
- **環境構築が不要**: Pythonの標準ライブラリ（Tkinter等）のみを使用しているため、追加の `pip install` なしですぐに動かせます。

---

## 🛠 動作環境

- Python 3.x

---

## 📦 使い方

リポジトリをクローンするか、ソースコードをダウンロードして実行してください。

### 1. CUI版の実行方法
ターミナルまたはコマンドプロンプトを開き、以下のコマンドを実行します。

```bash
python cui_app.py
画面の指示に従って、変換したい項目の番号（1〜4）と数値を入力してください。

2. GUI版の実行方法
以下のコマンドを実行すると、専用のデスクトップウィンドウが立ち上がります。

Bash
python gui_app.py
ドロップダウンメニューから変換したいモード（例: メートル ➔ フィート）を選択します。

入力欄に数値を入力します。

「変換する」ボタンを押すと、画面下部に結果が表示されます。

📝 コードの拡張について
新しい単位（例：温度の 摂氏 ⇄ 華氏 など）を追加したい場合は、以下の手順で簡単にカスタマイズできます。

変換ロジックの関数を追加する。

CUI版の場合は main() 内の条件分岐（if/elif）を追加する。

GUI版の場合は combo_mode の values リストと、convert() 関数内の条件分岐を追加する。
